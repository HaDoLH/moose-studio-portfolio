# 圖檔丟這裡就好

四個資料夾、檔名隨便取，丟好之後告訴 Claude「圖放好了」就會自動掛上網頁。

---

## 📁 hero/ — Hero 主視覺（網頁最上方）

丟 **3 張**：第一張是「主視覺大圖」（手機版只會顯示這張），其餘兩張是桌機版右側的小格。

- 建議比例：直式 3:4 或 4:5
- 內容建議：最有代表性的作品 → 例如 OKLAO 禮盒、PDF 封面那張咖啡濾杯

---

## 📁 peek/ — Sneak Peek 跑馬燈

丟 **8 張**（不用對應特定品牌，自動跑馬燈循環）。

- 建議比例：橫式 4:3 或 16:10
- 內容建議：你最想被看到的代表作

---

## 📁 works/ — 作品 4 大類別

丟 **4 張**，**檔名請帶上類別關鍵字**，幫助我判斷哪張對應哪一類：

- 檔名含「品牌」或 `brand` → 品牌視覺卡
- 檔名含「電商」或 `ecom` → 電商視覺卡
- 檔名含「展場」或 `exhibit` → 展場設計卡
- 檔名含「攝影」或 `photo` → 商業攝影卡

例如：`OKLAO禮盒_品牌.jpg`、`咖啡展場.jpg`、`霸丼天_攝影.jpg`

- 建議比例：橫式 16:10

---

## 📁 about/ — About 個人區

丟 **3 張**：工作室、設計過程、個人形象等。

- 第一張會變上方大圖（橫跨整列）
- 第二、三張會變下方左右小圖
- 建議比例：橫式 16:10 / 4:3

---

## 📐 格式提醒

- **副檔名**：`.jpg` 或 `.png` 都可以
- **大小**：長邊約 1600px 即可，太大網站會卡
- **不用一次全到位**：先丟一兩個資料夾、看完效果再補也行

---

## ✅ 丟完之後

跟 Claude 說：
- 「Hero 圖放好了」（只處理那一組）
- 或「全部都放好了」（一次處理全部）
